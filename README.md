# CyberGuard AI

## Overview

CyberGuard AI is a web based cybersecurity awareness and monitoring platform designed to provide users with insights into cyber threats, security practices, and safe digital behavior. The application aims to improve user awareness about online risks while offering an intuitive interface for interacting with cybersecurity related information.

This project was developed as part of an academic final project with emphasis on collaborative development, structured documentation, and version controlled contributions.

---

## Key Features

* User friendly cybersecurity awareness interface
* Threat awareness and informational content display
* Responsive frontend design for multiple screen sizes
* Fast performance using modern frontend tooling
* Clean modular code structure for maintainability
* Deployment ready build configuration

## Project Setup (Local Execution)

### Prerequisites

Make sure you have:

* Node.js (version 16 or above recommended)
* npm package manager
* Git installed

---

### Installation Steps

1. Clone the repository:

```
git clone https://github.com/<your-username>/CyberGuard-AI.git
```

2. Navigate into the project folder:

```
cd CyberGuard-AI
```

3. Install dependencies:

```
npm install
```

4. Start development server:

```
npm run dev
```

The application will run locally at:

```
http://localhost:5173
```

---

### Production Build

To generate production optimized files:

```
npm run build
```

Preview build locally:

```
npm run preview
```
